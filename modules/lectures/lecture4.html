Introduction To Array


An array is a collection of elements of the same data type stored in contiguous memory locations. Arrays are used to store and manipulate large amounts of data.


Characteristics of Arrays

  1. All elements in an array are of the same data type.
  2. Each element in an array is identified by an index or subscript.
  3. Arrays are stored in contiguous memory locations.
  4. Arrays have a fixed size that is specified when the array is created.

Types of Arrays

  1. One-dimensional array: A list of elements of the same data type.
  2. Multi-dimensional array: A table or matrix of elements of the same data type.

Declaring Arrays

  1. The syntax for declaring a one-dimensional array is: data_type array_name[array_size];
  2. The syntax for declaring a multi-dimensional array is: data_type array_name[array_size1][array_size2];.

Initializing Arrays

  1. Arrays can be initialized when they are declared using the syntax: data_type array_name[array_size] = {element1, element2, ...};
  2. Arrays can also be initialized using a loop.

Initializing Arrays

  1. Array elements can be accessed using their index or subscript.
  2. The syntax for accessing an array element is: array_name[index].

Exaples:

Example of a One-Dimensional Array:

int scores[5] = {90, 80, 70, 60, 50};

printf("%d", scores[0]); // prints 90


Example of a Multi-Dimensional Array:

int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};

printf("%d", matrix[0][1]); // prints 2

printf("%d", matrix[1][2]); // prints 6


Example of Accessing Array Elements:

int numbers[3] = {10, 20, 30};

printf("%d", numbers[1]); // prints 20

printf("%d", numbers[2]); // prints 30

printf("%d", numbers[0]); // prints 10


Example of Initializing Arrays Using a Loop:

int numbers[5];

for (int i = 0; i < 5; i++) {

numbers[i] = i * 10;

}
