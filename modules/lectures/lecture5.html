Introduction To Function


A function is a block of code that performs a specific task and can be called multiple times from different parts of a program. Functions are used to organize code, reduce repetition, and improve modularity.


Characteristics of Fuctions

  1. A function has a unique name that identifies it.
  2. A function takes arguments or parameters that are passed to it when it is called.
  3. A function returns a value to the calling program.
  4. A function can be called multiple times from different parts of a program.

Types of Functions

  1. Library functions: Pre-defined functions that are part of the programming language's standard library.
  2. User-defined functions: Functions that are defined by the programmer.

Declaring Functions

  1. The syntax for declaring a function is: return_type function_name(parameter_list) { function_body }
  2. The return_type specifies the type of value the function returns.
  3. The function_name is the unique name of the function.

Function Calling

  1. The syntax for calling a function is: function_name(argument_list);
  2. The argument_list specifies the values to be passed to the function.

Advantages of Functions

  1. Code reusability: Functions can be called multiple times from different parts of a program.
  2. Code modularity: Functions break down a large program into smaller, manageable modules.
  3. Easier maintenance: Functions make it easier to modify or update code.

Examples:

Example of a Function:

                                            int add(int x, int y) {

                                            return x + y;

                                            }

                                            int main() {

                                            int result = add(5, 3);

                                            printf("%d", result); // prints 8

                                            }


Example of a Library Function:

                                            printf("Hello, World!");

                                            scanf("%d", &num);

                                            strlen("Hello"); // returns 5

                                            rand(); // generates a random number

                                            exit(0); // exits the program



                                        
Example of a User-Defined Function:

                                            void greet() {

                                            printf("Hello, World!");

                                            }

                                            int main() {

                                            greet(); // prints Hello, World!

                                            }



                                        
Example of a Function with Arguments:

                                            }

                                            int main() {
